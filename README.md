# 歌手作词作曲 Agent Skills

华语流行歌手创作风格蒸馏库。每个歌手一个独立的 Agent Skill，加载后可让 AI 以该歌手的风格作词、作曲、编曲。

## 使用方式

### 加载 Skill

将歌手文件夹复制到你的 AI Agent skills 目录下。

**Claude Code:**
```
~/.claude/skills/
```

**Codex:**
```
~/.agents/skills/
```

### 触发示例

直接对 AI 说：

- "用周杰伦的中国风写一首关于'等待'的歌"
- "用陈奕迅的风格写一段怀念前任的歌词"
- "帮我给这段歌词配林俊杰式的和弦"
- "用陈绮贞的感觉写一首旅行主题的歌"

## Skill 章节结构

每个 Skill 包含 8 个章节：

| 章节 | 内容 |
|------|------|
| 使用说明 | 触发词 + 运行规则 + 硬性约束 |
| 认知层 | 心法、核心信念、情感基调矩阵、叙事视角 |
| 作词层 | 意象系统、句式模板、金句公式、语言DNA |
| 作曲层 | 旋律规则、节奏规则、和弦走向、编曲标志 |
| 创作模板 | 2-3个填空式创作框架（按曲风分） |
| 创作禁区 | 5-8条禁止项，每条附替代方案 |
| 对比示例 | 错误 vs 正确的完整对照 |
| 局限性 + 自检清单 | 样本限制 + 创作前 check list |

## 拓新歌手

1. 在 `_analysis/` 下写一份原始分析（参考现有 .md 的四段结构：作词→作曲→创作框架→练习建议）
2. 新建 `歌手名/` 文件夹，复制任一 SKILL.md 为模板
3. 按歌手的风格特点重填八章内容（特别注意禁区、对比示例——这是 personality 的核心）
4. 更新本 README 的收录表
