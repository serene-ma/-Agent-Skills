# 孙燕姿 创作指南

> 基于孙燕姿历年作品（华纳/环球时期）蒸馏 | 作词 + 作曲双维度 | 样本有限仅供参考

---

## 使用说明（运行规则）

**用孙燕姿验证过的方法论创作词曲：**

1. 用户说"用孙燕姿风格写一首关于 XX 的歌"时：
   - 先查**认知层**：这个话题她会持什么情感立场？用什么意象承载？
   - 再查**内容层**：用哪个句式模板？哪个和弦走向？节奏型怎么设计？
   - 最后查**模板层**：匹配慢歌还是快歌模板，按填空式框架输出

2. 用户说"帮我优化这段歌词/旋律"时：
   - 用内容层的句式公式检查主歌是否叙事、副歌是否有金句
   - 用作曲层的切分音规则检查旋律节奏是否有"姿式律动"
   - 用禁区清单逐条对照，砍掉违规内容

3. 用户说"给我 XX 主题的创作灵感"时：
   - 从意象系统出发推荐承载物（物件 > 情绪直述）
   - 从金句公式反推主歌叙事方向
   - 结合和弦走向推荐情感色彩

**硬性规则（优先级最高）：**
- 不能编造孙燕姿从未唱过的主题或态度
- 不能把通用作词技巧包装成"孙燕姿的独特方法"
- 所有公式和模板必须有原始作品作为来源
- 当用户问的话题超出孙燕姿风格覆盖范围时，明确说"这个方向不在孙燕姿风格范围内"

## 一、认知层 — 像孙燕姿一样创作

### 1.1 一句话心法

> **不要把情感"喊"出来，而是要把故事"讲"出来。用最干净的和弦，配最倔强的语气。**
>
> 我懂，我痛，但我选择放手 —— 这是"理性悲伤"的底色。

### 1.2 核心信念

**1. "悲伤不需要声嘶力竭，克制反而更戳人。"**
- 📍 证据：《开始懂了》"相信你只是怕伤害我，不是骗我"——把分手解读为善意谎言，高情商自我安慰
- 🎯 应用：写分手/离别类歌词时，禁止用"撕心裂肺""痛不欲生"等直白表述，改用理性视角重新解读伤害

**2. "具体的物件比抽象的情绪更有力量。"**
- 📍 证据：《隐形人》用"没织完的毛衣"替代"我很想你"；《天黑黑》用外婆唱歌的细节承载整个童年
- 🎯 应用：确立主题后，第一步不是写情感句子，而是找一个具体的承载物

**3. "旋律要像说话一样自然，最高音留给最想说的那个字。"**
- 📍 证据：《遇见》结尾叹气式下行音阶；《我怀念的》副歌最高点在"怀念"二字
- 🎯 应用：作曲时先朗读歌词，找到自然重音位置，旋律走向服从语言节奏

**4. "留白比填满更重要——换气口就是情感呼吸。"**
- 📍 证据：几乎所有慢歌都留出明显气口，段尾常用下行音阶制造"叹气感"
- 🎯 应用：旋律不要写太满，每句末留四分休止或更长，模拟说话时的自然停顿

### 1.3 情感基调矩阵

| 情感类型 | 孙燕姿处理方式 | 禁止方式 |
|---------|---------------|---------|
| 失恋/分手 | 理性解读、自我安慰、倔强释怀 | 哭诉、指责对方、自怨自艾 |
| 怀念/回忆 | 物是人非、细节触发、温暖中带酸涩 | 沉溺过去、后悔自责 |
| 成长/蜕变 | 经历风雨后的平静、接纳不完美 | 励志口号、成功学叙事 |
| 孤独/迷茫 | 画面感场景 + 轻描淡写的无助 | 绝望呐喊、黑暗渲染 |
| 自由/奔跑 | 切分节奏 + 风/天空意象 + 释放感 | 热血口号、对抗叙事 |

### 1.4 叙事视角

- **第一人称亲历者**：绝大多数歌曲。"我"是经历者，不是旁观者
- **对"你"说话**：第二人称对话感强，像写信或自言自语
- **不审判、不说教**：即使对方"错"了，也不在歌词里定罪
- **创作时代入的角色**："你是一个温柔但倔强的叙述者。你经历过、你还在痛、但你选择往前走。你不会大喊大叫，你会把故事讲给一个老朋友听。"

## 二、作词层 — 像孙燕姿一样写词

### 2.1 意象系统（物件 > 情感）

| 类别 | 核心意象 | 来源作品 | 情感功能 |
|------|---------|---------|---------|
| 天空系 | 风筝、风、云、天空、雨天 | 《风筝》《雨天》《天黑黑》 | 距离感、不可控、等待 |
| 身体系 | 眼泪、背影、眼睛、声音、手 | 《开始懂了》《我怀念的》 | 亲密残留、无法触碰 |
| 场所系 | 十字路口、咖啡店、老地方 | 《雨天》《隐形人》 | 选择困境、回忆锚点 |
| 物件系 | 毛衣、脚踏车、信、日记 | 《隐形人》 | 未完成的承诺 |
| 时间系 | 天黑、天黑黑、那一天、很久以前 | 《天黑黑》 | 转折点、回不去的节点 |

**意象选择原则：**
- 一件具体的物件 > 十个抽象形容词
- 同一个意象在副歌至少复现一次，形成记忆锚点
- 自然意象（风/雨/云）承载情绪，人造物（毛衣/信/咖啡）承载故事

### 2.2 句式结构模板

**主歌（A段）：陈述句叙事**

```
句式：我 + [具体动作] + [具体场景/物件] + [隐含情绪]

例句拆解：
- "我 走在 那个转角咖啡店" + "点了你常喝的拿铁" → 没有说"我想你"，但你懂了
- "我 看着 那件没织完的毛衣" + "线头还留在针上" → 没有说"遗憾"，但你感受到了
```

**规则**：主歌每句必须包含一个可感知的**具体细节**（看到的/听到的/触碰到的），不允许纯情绪句。

**预副歌（B1段）：一句话转折**

```
句式：[时间词] + [才发现/才明白/才开始] + [反转认知]

例句：
- "原来人会变得温柔，是透彻的懂了" ——《开始懂了》
- "我还不习惯，在没有你的房间" ——《隐形人》
```

**规则**：预副歌完成一次认知升级，从叙事过渡到感悟。不超过两句。

**副歌（B2段）：感叹句金句**

```
句式：[可是/但是/然而] + [我 + 核心动词] + [的] + [是] + [情感名词/金句]

例句：
- "可是 我怀念的，是无话不说" ——《我怀念的》
- "但是 我会记得，那年冬天" ——《我的爱》
```

**规则**：副歌第一个字必须是转折连词，制造情感张力释放。最高音必须落在最核心的情感词上。

### 2.3 金句公式

| 公式 | 结构 | 例句 | 使用场景 |
|------|------|------|---------|
| 反转解读 | 把负面事件重新解释为正面意图 | "相信你只是怕伤害我，不是骗我" | 分手/离别 |
| 时间对比 | [过去] vs [现在]，物是人非 | "以前是以前，现在是现在" | 怀念/成长 |
| 条件让步 | 我可以接受X，只要Y | "我不要将你多绑住一秒" | 放手/自由 |
| 最小单位 | 缩小到最小的情感颗粒 | "我怀念的是无话不说 / 我怀念的是争吵以后" | 回忆细节 |

**金句创作流程：**
1. 确定情感主题（如：舍不得但必须放手）
2. 用"反转解读"公式写一个候选金句
3. 检查：这句话能发朋友圈吗？能被引用吗？
4. 不能 → 说明太复杂，回到步骤2换一个更简洁的角度

### 2.4 语言DNA

- **口语化程度**：高。像在对朋友说话或写日记，允许不完整的句子
- **人称策略**：我 > 你 > 他/她。视角始终在第一人称
- **句式节奏**：短句为主，8-12字/句；副歌单句不超过15字
- **禁用词**：撕心裂肺、痛不欲生、永远、一辈子（太绝对）、你知道（说教感）
- **高频结构词**：原来、可是、但是、却、才、就、也
- **签名句式**："我 + [动词] + 的" 结构（我怀念的 / 我想要的 / 我相信的）

## 三、作曲层 — 像孙燕姿一样谱曲

### 3.1 旋律设计规则

**规则 1：自然音阶为主，避免过度转音**
- 旋律走向符合说话音调，像"朗读有感情的台词"
- 禁止大量 R&B 式转音、花腔、装饰音
- 相邻音程以二度/三度为主，大跳（≥五度）仅用于副歌情感爆发点

**规则 2：音域控制在中高音区**
- 主歌：C4 - G4（真声舒适区）
- 副歌：E4 - C5（弱混声舒适区，最高音仅出现1-2次）
- 整体控制在一个八度 + 纯四度以内
- 禁止持续高音区徘徊（会让听众觉得"这歌我唱不了"）

**规则 3：最高音 = 最想说的字**
- 副歌最高音必须落在情感核心词上
- 如《我怀念的》最高音在"怀念"；《遇见》最高音在"未来"

**规则 4：段尾叹气式下行**
- 每个乐段结尾做二度或三度下行音阶
- 模拟说话结束时自然的音调下降
- 代表作：《遇见》"我排着队拿着爱的号码牌"末尾下行

### 3.2 节奏设计规则（姿式律动核心）

**慢歌（抒情）：**
- 主歌：四分 + 八分音符交替，一字一音为主
- 副歌：在关键词位置引入**三连音**或**切分音**，制造哽咽感
- 规则：把三个字挤在两个拍子里 → 呼吸急促/感情涌上的效果
- 段尾：留出至少半拍到一拍的气口（休止符或长音延续）

**快歌（摇滚/轻快）：**
- 大量使用**反拍重音**和**切分节奏**，制造"慵懒中带着冲劲"的奔跑感
- 参考：《第一天》《超快感》——弱拍进、切分节奏驱动
- 规则：重音不在正拍上，而是落在拍子之间的"缝隙"里
- 禁止四平八稳的军鼓式节奏

**节奏禁忌：**
- 慢歌禁止一字两音以上的花腔（那不是孙燕姿）
- 快歌禁止从头到尾正拍强音（破坏松弛感）
- 禁止连续十六分音符（太紧张、不够从容）

### 3.3 和弦走向

**基础进行（慢歌首选）：**

```
调性：C大调 / G大调（偏好明亮中带温暖的调性）

核心走向 1（卡农进行）：
I - V - vi - iii - IV - I - IV - V
C - G - Am - Em - F - C - F - G
代表作：《我怀念的》《开始懂了》

核心走向 2（色彩变化）：
I - V - vi - IV（简化四和弦）
C - G - Am - F
在段落衔接处使用

色彩技巧：
- 关键情感位置插入 挂留和弦（sus4/sus2），营造"悬而未决"感
- 副歌进入前使用 V7 → I 的解决，增强释放感
- 桥段可短暂离调到 iv 小调和弦，制造"回忆涌上心头"的暗色
```

**快歌进行：**
```
多用 I - IV - V 的经典摇滚框架
加入 bVII（降七级）制造松弛感
节奏吉他用切分拨弦替代正拍扫弦
```

### 3.4 人声当作乐器

- **旋律密度**：不要写得太满。每句之间留呼吸空间
- **气口设计**：在每句歌词的自然断句处设换气点，这是情感传递的关键
- **动态起伏**：主歌平稳（mp）、预副歌渐强（mf）、副歌释放（f）、段尾渐弱（dim.）
- **禁止**：连续高音咆哮（那是摇滚，不是姿式）、急促换气（说明旋律太密）

## 四、创作模板

### 模板 A：姿式慢歌（抒情类）

```
【第一步：抓取物件】
选一个具体而微的东西（毛衣/咖啡杯/脚踏车/路灯/旧照片）
→ 这就是你的意象锚点

【第二步：主歌叙事（A1-A2）】
A1：描写当下的场景 + 那个物件出现了
    "我 + [动作] + [地点] + [看到/碰到] + [物件]"
A2：物件引发的回忆细节（2-3个具体画面）
    "那天 + [细节1] / [细节2] / [细节3]"

【第三步：预副歌转折（B1）】
一句话认知反转："[时间词] + 才[发现/明白/知道] + [感悟]"
→ 从叙事切换到感情

【第四步：副歌金句（B2）】
"可是/但是 + 我[核心动词]的 + 是 + [金句]"
→ 最高音放在金句的情感词上
→ 然后叹气式下行收尾

【第五步：和弦编排】
C大调 → I-V-vi-iii-IV-I-IV-V
预副歌进入前用 V7 → 副歌第一拍落在 I
副歌关键位置插入 sus2/sus4

【第六步：节奏编排】
主歌：四分+八分，一字一音
副歌关键词：三连音（三个字挤两拍）
段尾：一拍以上休止
```

### 模板 B：姿式快歌（轻摇滚/奔跑类）

```
【第一步：确立态度】
选一种释放感的场景（奔跑/出发/离开/飞翔）
→ 关键词：风、速度、不要回头

【第二步：主歌叙事（A段）】
用反拍节奏唱出"正在做某事"的状态
每句从弱拍进入，重音落在"的缝里"

【第三步：副歌释放（B段）】
切分音驱动 + 简短有力的句子
"我 + [动词] + [目的地/方向]"
音域比主歌高纯四度左右

【第四步：和弦编排】
C大调 → I-IV-V 为主
加入 bVII 制造松弛感
节奏吉他切分拨弦

【第五步：尾声处理】
突然安静或 fade out
对比感 > 完美收束
```

## 五、创作禁区

**以下行为会让作品失去孙燕姿风格的核心竞争力：**

1. **禁止：把情感直接"喊"出来**
   - "我很伤心""我好想你""我真的很痛"——这是禁区
   - 替代方案：写"那杯咖啡凉了，和那天一样"

2. **禁止：副歌没有金句**
   - 姿式慢歌的副歌必须有1-2个"能被记住、能被引用"的句子
   - 检验标准：这句话能单独发朋友圈吗？不能 → 重写

3. **禁止：旋律填满每个拍子**
   - 无气口 = 无呼吸 = 无情感。每句末尾必须留白
   - 检验标准：跟着哼一遍，中间有自然的换气停顿吗？

4. **禁止：音域超出普通人舒适区**
   - 最高音只放在最关键的情感词上，出现不超过2次
   - 禁止副歌全程在高音区徘徊

5. **禁止：和弦走向太"爵士"或太"复杂"**
   - 孙燕姿的和弦以干净为主，色彩和弦只是点缀
   - 如果一个和弦你不知道为什么要加，就删掉

6. **禁止：在歌词里审判对方**
   - "你骗了我""你辜负了我"——这是孙燕姿永远不会写的话
   - 即使对方有错，也要用理解的口吻

7. **禁止：堆砌华丽辞藻**
   - 姿式歌词不是说唱、不是古风、不是现代诗
   - 是"干净的口语"，像在和朋友说一件真实发生过的事

## 六、对比示例

### 示例 1：写分手（错误 vs 正确）

❌ **错误写法（撕心裂肺型）：**
```
你为什么离开我
我的心像被撕裂了一样痛
没有你的日子我活不下去
回来吧，我不能没有你
```
> 问题：直接"喊"情感、审判对方、没有意象承载

✅ **姿式写法：**
```
[主歌 A1]
我走到那个转角咖啡店
点了你常喝的拿铁
糖包还放在老位置
只是这次没人会用

[预副歌]
原来习惯比爱情更诚实
不需要誓言也能证明

[副歌]
可是我怀念的 不是你在身边
是你推门时 风铃响起的瞬间
我承认还痛 但不需要抱歉
至少那杯拿铁 还是热的
```
> 改进：咖啡店/拿铁/糖包/风铃——四个具体物件承载全部情感；副歌金句"风铃响起的瞬间"是可被记住的画面；"还痛但不需要抱歉"是理性的倔强

### 示例 2：写成长（错误 vs 正确）

❌ **错误写法（鸡汤型）：**
```
我终于变得强大
感谢所有伤害过我的人
从今以后 我会活得更好
让过去都过去吧
```
> 问题：说教口吻、空洞口号、毫无画面

✅ **姿式写法：**
```
[主歌]
那件没织完的毛衣 还在衣柜最里面
线头散了一半 像停在某个冬天
```

## 八、局限性 + 自检清单

### 8.1 局限性声明

1. **样本限制**：本指南基于孙燕姿华纳/环球时期主要作品分析，未覆盖全部曲目。
2. **风格边界**：孙燕姿曲风多变（抒情/摇滚/轻快/电子），本指南偏向其最具辨识度的"姿式慢歌"风格。
3. **时代差异**：早期作品（2000-2007）与后期作品在制作风格上有差异，本指南以中期"姿式"巅峰期为基准。
4. **个体局限**：孙燕姿的独特音色和气息控制是声乐天赋，无法通过文字指南完全传达。

### 8.2 创作前自检清单

在用这个 skill 创作前，先确认：

- [ ] 我找到了至少一个具体的物件作为意象锚点（不是抽象概念）
- [ ] 主歌每句都有可感知的细节（看到/听到/触碰到的）
- [ ] 副歌第一句是转折连词开头（可是/但是/然而）
- [ ] 副歌有1-2个"能发朋友圈"的金句
- [ ] 旋律最高音落在歌词最核心的情感词上
- [ ] 每个乐段末尾留了气口/下行收尾
- [ ] 和弦走向干净，没有无理由的复杂和弦
- [ ] 歌词中没有直接"喊"情感（很痛/好想你/太伤心）
- [ ] 没有审判、说教、或过于绝对化的表述
- [ ] 用人声试唱一遍，旋律是否像"说话"一样自然
