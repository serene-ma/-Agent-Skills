# 林俊杰 创作指南

> 基于林俊杰历年作品分析 | 作词 + 作曲双维度 | 行走的CD级旋律工艺

---

## 使用说明

**用林俊杰的方法论创作：**

1. 用户说"用林俊杰风格写一首XX的歌"时：
   - 先确定情感基调：JJ的歌永远是"温暖的"。即使是悲伤的歌，底色也是理解和治愈，不是冷和绝望
   - 作曲优先：JJ的歌是旋律驱动的，先有好旋律，歌词服务于旋律的情感走向
   - 副歌必须"打开"——从一个中低音的温柔叙述，突然跳到高音区的全力拥抱

2. **硬性规则：**
   - 副歌必须有至少一次大幅音域跨越（五度以上跳进）
   - 旋律必须有转音/装饰音——但控制在"好听不炫技"的范围内
   - 和弦必须有爵士色彩（maj7/min9/add9 至少一种）

## 一、认知层 — 林俊杰的音乐哲学

### 1.1 一句话心法

> **旋律要会呼吸。用最温柔的转音包裹最痛的话，用最高的音唱出最暖的拥抱。**

### 1.2 核心信念

**1. "旋律是第一位的。好的旋律不需要歌词也能让人哭。"**
- 📍 证据：JJ的作品经常先有demo旋律，后填词。旋律本身就包含完整的情感弧线
- 🎯 应用：创作时先确立旋律的核心动机（motif），围绕这个动机展开全曲

**2. "治愈不是粉饰太平，是在伤口上放一片温暖的羽毛。"**
- 📍 证据：《修炼爱情》《可惜没如果》——直面遗憾和痛苦，但用温暖的旋律包裹
- 🎯 应用：歌词可以写痛，但旋律必须是暖的。大调占比 > 小调

**3. "最高音必须放在最动人的那个字上，因为它值得。"**
- 📍 证据：几乎所有副歌高潮点，最高音都精准落在情感核心词上
- 🎯 应用：确定副歌核心词 → 为它预留最高音 → 前面的旋律为此铺垫

**4. "钢琴就是我的语言。和弦走向要像手指在琴键上自然流淌。"**
- 📍 证据：JJ坐在钢琴前即兴的旋律，往往成为传世之作
- 🎯 应用：创作时想象手指在钢琴上的运动——旋律走向要"顺手"

### 1.3 情感基调矩阵

| 情感类型 | 林俊杰处理方式 | 禁止方式 |
|---------|---------------|---------|
| 遗憾/错过 | 温暖回望 + "如果"句式 + 修炼/成长视角 | 指责、后悔、绝望 |
| 暗恋/守护 | 默默陪伴 + 不打扰的温柔 | 占有欲、强求 |
| 梦想/坚持 | 信念感 + 宏大的副歌释放 + "光"意象 | 空洞口号 |
| 孤独/迷茫 | 被理解的渴望 + 最终的和解 | 黑暗沉溺 |
| 友情/感恩 | 温暖叙事 + 具体细节 | 煽情过度 |

### 1.4 叙事视角

- **"我"是叙述者，也是治愈者**：即使是写自己的痛，也有一种"我在陪着你（包括陪我自己）"的温暖
- **对"你"永远温柔**：即使是分手的对象，也用"修炼""祝福""感谢"的口吻
- **创作时代入的角色**："你是一个温柔但有力的歌者。你的声音能照亮黑暗、接住坠落的人。你不会嘲笑脆弱，因为你自己也有。你不喊，但你唱。"

## 二、作词层

### 2.1 意象系统（JJ式温柔）

| 类别 | 核心意象 | 来源 | 情感功能 |
|------|---------|------|---------|
| 光系 | 光、黎明、阳光、星光 | 《不为谁而作的歌》 | 希望、治愈 |
| 水系 | 雨、江南烟雨、眼泪、海 | 《江南》《修炼爱情》 | 思念、清洗、流淌 |
| 时间系 | 冻结、一千年以后、很久以前 | 《冻结》《一千年以后》 | 跨越时间的爱 |
| 空间系 | 江南、赤道、星球、天堂 | 《江南》 | 距离感、遥远的美 |
| 身体系 | 心跳、声音、眼睛、手 | 《她说》《修炼爱情》 | 温柔的连接 |

**意象选用原则：**
- "光"和"水"是JJ最核心的两个意象系
- 光的意象 → 希望/治愈类主题
- 水的意象 → 思念/遗憾类主题
- 可以交叉使用：光穿过水 → "雨后的阳光"

### 2.2 句式模板

**主歌：温柔铺垫**
```
句式：一个场景 + 一个内心独白 + 一个未完成的句子

例："凭什么要失望，藏眼泪到心脏" ——《修炼爱情》
   "雨到这里缠成线，缠着我们留人自己的泪" ——《江南》
```

**预副歌：节奏加密推进**
```
句式：三连音/十六分音符快速过度的"桥接句"
→ 制造"要发生什么了"的紧张感
→ 为副歌的音域跳跃做铺垫
```

**副歌：全力释放（林氏爆发）**
```
句式：[动词 + 宾语] + [的 + 情感名词] + [温暖结论]

例："修炼 + 爱情 + 的心酸" ——《修炼爱情》
   "不为 + 谁 + 而作的歌" ——《不为谁而作的歌》

规则：
- 副歌第一句必须包含核心hook词（通常就是歌名）
- 最高音精准落在这个hook词上
- 旋律线呈"上扬弧线"——像拥抱打开
```

### 2.3 金句公式

| 公式 | 结构 | 例句 | 场景 |
|------|------|------|------|
| 修炼型 | [动词] + 爱情/幸福/梦想 + 的 + [感悟] | "修炼爱情的心酸" | 遗憾/成长 |
| 如果不型 | 可惜 + 没如果 / 如果 + 假设 | "可惜没如果" | 后悔/错过 |
| 不为谁型 | 不为 + [对象] + 而 + [做] | "不为谁而作的歌" | 自我/信念 |
| 冻结型 | 把 + [美好的东西] + 冻结/定格 | "把爱冻结在那一秒" | 珍惜/怀念 |
| 她说型 | 她说 + [一句话] | "她说无所谓，只要能在夜里翻来覆去" | 第三人称叙事 |

### 2.4 语言DNA

- **口语化程度**：中等。比周杰伦"自己写"的雅一点，比方文山"华丽"的俗一点
- **禁用词**：恨、报复、诅咒、绝望、永远不再——JJ的世界没有恨
- **高频词**：爱、光、修炼、如果、心跳、梦、等、她说、记得
- **签名句式**："[动词] + 爱情" 结构（修炼爱情/冻结爱情/形容爱情）

## 三、作曲层

### 3.1 旋律设计

**规则 1：音域从低到高的"弧线型"**
- 主歌：中低音区（C4-E4），温柔叙述
- 预副歌：升key推进（F4-G4），节奏加密
- 副歌：高音区释放（A4-C5），最高音在hook词上
- 典型弧线：___／‾‾‾＼___（主歌平 → 预副歌推 → 副歌爆发 → 回落）

**规则 2：R&B转音套娃**
- 在长音上做2-3个音符的快速装饰（turn/mordent）
- 转音位置：副歌每个长音的前半拍 → 后半拍做装饰
- **铁律**：每个长音最多转2-3次（多了叫炫技，少了叫不够）

**规则 3：旋律解决到"温暖"的主音**
- 即使是小调的歌，最后一句也倾向解决到大调的平行主音
- 例：A小调的歌曲，最后停在C大调的C上——"虽然痛，但还有光"

### 3.2 节奏设计

| 段落 | 节奏特征 | 作用 |
|------|---------|------|
| 主歌 | 平稳四分+八分，钢琴柱式和弦推进 | 建立叙述空间 |
| 预副歌 | 节奏密度翻倍（八分→十六分或三连音） | 制造推进感 |
| 副歌 | 长音释放 + 附点节奏 | 情感释放 |
| 桥段 | 突然安静 + 不规则留白 | 呼吸/反思 |

**JJ式节奏特点**：
- 不像周杰伦的"反拍歪"，更偏R&B的正拍摇摆（laid-back feel）
- 钢琴伴奏有爵士的swing感
- 副歌不赶、不急——高音拖够，让听众"沉浸"进去

### 3.3 和弦走向

**核心走向 1：温暖叙事型**
```
I - V/7 - vi - iii - IV - I/3 - ii7 - V7
C - G/B - Am - Em - F - C/E - Dm7 - G7
（大量使用转位和弦，低音线走旋律）
```

**核心走向 2：JJ式大歌型**
```
I - V - vi - IV（四和弦循环）
但每个和弦都升级为色彩和弦：
Cmaj9 - G7sus - Am9 - Fmaj7
```

**关键色彩技巧：**
- **maj9 / add9**：基础款，制造"温暖"感——JJ式暖色滤镜
- **m7 / min9**：在"遗憾"段落使用，苦涩但柔和
- **转位和弦**：低音线走出副旋律，这是JJ钢琴思维的标志
- **远关系转调**：桥段/最后一次副歌直接升半音或全音，制造"升维"感

### 3.4 编曲标志

- **钢琴绝对主导**：一切从钢琴出发。即使加了弦乐和鼓，钢琴始终是骨架
- **弦乐在副歌进入**：不是一开始就铺满，副歌才进——"该来的时候来"
- **鼓点简洁**：慢歌用简单的底鼓+军鼓，不抢旋律风头
- **前奏偏长**：8-16小节钢琴独奏建立情绪（这是JJ的习惯）
- **Bridge的"安静时刻"**：桥段往往突然抽走所有乐器只剩钢琴+人声

## 四、创作模板

### 模板 A：JJ式治愈情歌

```
【作词】
主歌：选一个"水"或"光"的意象 → 写一段温柔但有心事的叙述
预副歌：节奏加密的推进句（"我拼命/我努力/我学着..."）
副歌：hook词（歌名） + 最高音爆发 + 温暖的和解句
结尾：一个"光"的意象收尾

【作曲】
调性：C大调或G大调（明亮的）
和弦：maj9化处理的全套
旋律：主歌 C4-E4 → 预副歌 F4-G4 → 副歌 G4-C5
转音：副歌每个长音记着加2-3音装饰
```

### 模板 B：JJ式信念大歌

```
【作词】
定一个"光/梦想/爱"的大主题
主歌写"为什么需要这首歌"（一个具体的痛/迷茫）
副歌写"这首歌要给你什么"（勇气/信念/温暖）

【作曲】
调性：D大调或E大调（更有力量）
和弦：I-V-vi-IV循环，maj9+add9
最后一次副歌升半个到一个全音（升维）
结尾主和弦长音fade out
```

### 模板 C：JJ式R&B轻快

```
【作词】
选择一个轻松主题（心动/阳光/在一起）
轻快的日常场景，不要沉重

【作曲】
节奏有swing/laid-back feel
人声旋律在拍子后面一点点（laid-back唱法）
和弦多m7，少maj7（轻松 > 深沉）
副歌hook重复洗脑
```

## 五、创作禁区

1. **禁止：写恨**
   - JJ的歌里没有恨。遗憾可以，但要用"修炼"的角度，不能怪对方

2. **禁止：旋律没有高潮**
   - 副歌必须有大跳（五度以上），不能主歌和副歌在同一个音区徘徊

3. **禁止：和弦太"干"**
   - 三和弦的JJ不叫JJ。至少要加七音，最好加九音或sus

4. **禁止：转音过多**
   - JJ是"有分寸的华丽"。每个长音转2-3次即可。超过5次 = 炫技病

5. **禁止：Bridge不收住**
   - 桥段必须先"安静"再"重新燃起"，不能全程高能

6. **禁止：副歌hook不清晰**
   - 听众必须能在第一次听就记住歌名/核心hook词

7. **禁止：用悲观的结尾**
   - JJ的歌最后一定是"有光"的。即使遗憾，也是"谢谢你让我成长"

## 六、对比示例

### ❌ 错误：冷感情歌
```
你走后世界变成灰色
我的心像被刀割开
为什么你要这样对我
我恨你的所有一切
```
> 问题：充满恨意、没有温暖、和弦想象一下就是冰冷的小三和弦循环

### ✅ JJ式写法：
```
[主歌]
雨停了 窗外的光慢慢变亮
我还没习惯 右手边空荡荡
你说的话 还在空气里回荡
我学着把它 变成感谢的重量

[预副歌]
曾经拼命想抓住的那些 现在学会了放手
不是不难过 是懂得

[副歌]
修炼爱情的心酸 学会放好以前的渴望
你给的温暖 还在血液里流淌
谢谢你 让遗憾 变成了光
```
> 感谢 > 恨、光 > 黑暗、修炼 > 沉溺

## 八、局限性 + 自检清单

### 局限性
1. JJ的唱功（音域/转音/气息）是天赋级，创作指南只能接近"JJ会怎么写"，无法替代他的演唱
2. 本指南偏向"林氏大情歌"风格（最具有辨识度的主线），对电子/实验作品覆盖有限

### 自检清单
- [ ] 歌词中有恨、诅咒、绝望吗？→ 有 = 重写
- [ ] 副歌有至少五度以上的音域跳进吗？
- [ ] 副歌hook词（歌名）放在最高音上了吗？
- [ ] 和弦有maj9/m7/add9等色彩和弦吗？
- [ ] 转音控制在2-3音/长音了吗（没有炫技）？
- [ ] 结尾是温暖/有光的吗？
- [ ] Bridge有"安静→再燃"的起伏吗？
