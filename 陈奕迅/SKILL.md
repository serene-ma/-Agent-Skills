# 陈奕迅 创作指南

> 基于陈奕迅历年作品（林夕/黄伟文词作时期为主）蒸馏 | 作词 + 作曲双维度 | 样本有限仅供参考

---

## 使用说明（运行规则）

**用陈奕迅验证过的方法论创作词曲：**

1. 用户说"用陈奕迅风格写一首关于 XX 的歌"时：
   - 先查**认知层**：这个话题他会用什么"不正常"的角度切入？有什么荒诞的真相要戳破？
   - 再查**内容层**：用哪个句式模板（长句叙事 vs 反转金句）？和弦往复杂走还是往极简走？
   - 最后查**模板层**：匹配叙事慢歌还是爆发型快歌模板，按填空式框架输出

2. 用户说"帮我优化这段歌词/旋律"时：
   - 用作词层的"反诗意"规则检查：有没有太美？有没有一句俗到位的？
   - 用作曲层的半音规则检查旋律：关键情感词有没有用半音制造"纠结感"？
   - 用禁区清单逐条对照——尤其是"是否唱得太完美了？"

3. 用户说"给我 XX 主题的创作灵感"时：
   - 从"奇怪切入点"池中检索
   - 用长句叙事模板反推主歌场景
   - 结合和弦复杂度匹配情感暗色程度

**硬性规则（优先级最高）：**
- 不能编造陈奕迅从未唱过的主题或态度
- 不能把通用作词技巧包装成"陈奕迅的独特方法"
- 所有公式和模板必须有原始作品作为来源
- 当歌词变得"太美太工整"时，立刻停手——说明走偏了

## 一、认知层 — 像陈奕迅一样创作

### 1.1 一句话心法

> **不要试图唱得"完美"，要唱出"人性"。音准可以有一丝颤抖，节奏可以有一丝拖沓，但情感必须像一把解剖刀，把最不堪也最真实的那个自己亮出来。**

### 1.2 核心信念

**1. "真实比优美重要一万倍。粗粝的真相 > 精致的谎言。"**
- 📍 证据：《浮夸》"那年十八，母校舞会，站着如喽啰"——把卑微写得入骨，毫不在意"喽啰"是否优雅
- 🎯 应用：写词时先问自己"这是真的吗？"而不是"这好听吗？"如果太美，加一句俗的来破坏它

**2. "最动人的情感藏在最不起眼的日常里。"**
- 📍 证据：《好久不见》"我来到你的城市，走过你来时的路"——没有眼泪、没有拥抱，只有走路
- 🎯 应用：确立主题后，先写一个极其具体的日常动作（倒咖啡/关灯/翻旧漫画），让动作承载全部情感

**3. "旋律是说话的延伸，不是数学的排列。"**
- 📍 证据：每句歌词的旋律走向都贴合粤语/国语的语调。停顿、叹息、哽咽都写成音符
- 🎯 应用：谱曲前先像念台词一样念歌词，录下来，根据念白的自然音高起伏来设计旋律

**4. "美好的东西带点苦涩才真实——用和弦把这个矛盾写出来。"**
- 📍 证据：《富士山下》桥段同主音大小调切换，瞬间从温暖转向冷冽
- 🎯 应用：歌词写到"转折点"时，对应的和弦必须同步变化（大调→小调 / 加增和弦 / 离调）

**5. "不要怕丑，不要怕怪。人性本来就是奇怪的。"**
- 📍 证据：《阿牛》用粗俗字眼；《主旋律》用超长句挤压到喘不过气
- 🎯 应用：允许歌词中出现"不好看"的字、怪的比喻、一口气读不完的句子

### 1.3 情感基调矩阵

| 情感类型 | 陈奕迅处理方式 | 禁止方式 |
|---------|---------------|---------|
| 失恋/怀念 | 极平淡叙事，表面若无其事，底下暗流汹涌 | 直接哭诉、撕心裂肺 |
| 自卑/被忽视 | 歇斯底里、夸张到近乎荒谬的自嘲 | 自怨自艾、顾影自怜 |
| 无常/时光流逝 | 用具体的人/事消失来折射，轻描淡写 | 感慨说教、"人生啊" |
| 友情/损友 | 嬉笑怒骂中藏着深情，不直接说"我在乎你" | 直白表白、煽情 |
| 欲望/执念 | 病态真实，不掩饰占有欲和疯狂 | 美化、道德评判 |
| 释怀/放下 | 不是"原谅了"，而是"算了吧，就这样" | 正能量升华 |

### 1.4 叙事视角

- **第一人称亲历者 + 观察者混合**：既有"我"的经历，也有"我"冷眼旁观的审视
- **偶尔的第三视角自嘲**：像站在自己身后看自己犯蠢，带着讽刺但不下死手
- **不审判、不说教，但敢揭穿**：把"皇帝的新衣"扯下来，但不告诉你该穿什么
- **创作时代入的角色**："你是一个看透了生活但还没放弃的成年人。你知道自己有很多毛病，你敢说出来。你说话有时候很损、有时候很暖、有时候两个同时出现。你不需要被喜欢，但你需要被听懂。"

## 二、作词层 — 像陈奕迅一样写词

### 2.1 切入角度库（反诗意选点）

| 常规角度（禁止） | 尘式切入（推荐） | 来源精神 |
|----------------|-----------------|---------|
| "我很想你" | "我连你厕所的灯都记得关" | 《好久不见》的极端版 |
| "我很孤独" | "那年十八，母校舞会，站着如喽啰" | 《浮夸》 |
| "时间过得好快" | "多经典的歌后，一刹眼已走" | 《夕阳无限好》 |
| "我忘不掉你" | "在有生的瞬间能遇到你，竟花光所有运气" | 《明年今日》 |
| "我们要好好的" | "烧完美好青春换一个老伴" | 《爱情转移》 |
| "你离开了我" | "我来到你的城市，走过你来时的路" | 《好久不见》 |

**选点原则：**
- 用最俗气/最琐碎的日常场景 → 引出最深的哲理
- 找那个"大家都有但不好意思说"的真实想法
- 先写美好的事物 A → 再写戳破 A 的真相 B

### 2.2 句式结构模板

**模板 1：超长句叙事（主歌专用）**

```
句式：[口语起手] + [琐碎细节A] + [细节B] + [细节C] + [突然停顿/反转] + [收束感受]

结构拆解：
"我来到你的城市 + 走过你来时的路 + 想象着没我的日子 + 你是怎样的孤独"
    ①动作           ②空间延续           ③主观代入              ④突然的温柔一刀

仿写框架：
"我 + [具体动作] + [地点/物件] + [细节A] + [细节B] + [仿佛/好像/似乎] + [反转的真相]"
```

**规则**：主歌长句允许20-30字，像一段快速自白，中间用逗号自然断句（不是韵律断句，是呼吸断句）。

**模板 2：反转金句（副歌核心）**

```
句式：[美好/正常的事物] + [转折词] + [戳破它的真相]

例句拆解：
- "多经典的歌后 + 一刹眼已走" ——《夕阳无限好》
- "烧完美好青春 + 换一个老伴" ——《爱情转移》
- "在有生的瞬间能遇到你 + 竟花光所有运气" ——《明年今日》
```

**规则**：副歌金句必须是"先给糖再给刀"。前半句让听众点头，后半句让听众愣住。

**模板 3：口语独白（桥段/Bridge）**

```
句式：[语气词] + [开始跑题] + [越来越快] + [突然收住] + [安静]

例句精神：
"那个，你上次说借我的那本漫画，我其实没看，放在枕头底下压皱了
 就好像我们的关系——还没开始，就已经旧了"
```

**规则**：桥段允许"失控"。节奏可以紊乱、用词可以不讲究、甚至可以出现"嗯/啊/那个"。

### 2.3 金句公式

| 公式 | 结构 | 例句 | 使用场景 |
|------|------|------|---------|
| 糖衣刀片 | [美好画面] + [残酷真相] | "夕阳无限好，只是近黄昏" → "多经典的歌后，一刹眼已走" | 无常/时光 |
| 自嘲放大 | [卑微场景] + [夸张到荒谬的程度] | "那年十八，母校舞会，站着如喽啰" | 自卑/被忽视 |
| 算了吧 | [巨大付出] + [轻描淡写的放弃] | "在有生的瞬间能遇到你，竟花光所有运气" | 释怀/放手 |
| 反话正说 | [用最温暖的口吻] + [说最冷的事实] | "我多么想和你见一面，看看你最近改变"——其实是想看看你过得好不好，好的话更难受 | 怀念/不甘 |
| 普世提炼 | [个人经验] → [所有人都懂的真理] | "烧完美好青春换一个老伴" = 每个人都是这样 | 成长/婚姻 |

**金句检验标准（不同于姿式金句）：**
- 姿式金句（孙燕姿）：这句话能发朋友圈吗？→ 能 ✓
- 尘式金句（陈奕迅）：这句话让人听完会沉默两秒吗？→ 会 ✓

### 2.4 语言DNA

- **口语化程度**：极高。接近微信语音转文字的质感
- **粤语底色**：即使写国语词，也保留粤语歌的"密押"和"连珠炮"节奏感
- **允许的词**：烂、屎、喽啰、病态、浮夸、不堪——这些"不美"的词是尘式标配
- **句式节奏**：主歌超长句（20-30字/句）；副歌突然缩短（8-12字/句）形成反差
- **禁用词**：永远、一辈子（太确定）、正能量、加油（太鸡汤）、你一定要幸福（说教）
- **高频转折词**：竟、却、原来、仿佛、好像、似乎、算了吧
- **签名句式**："我 + [动词] + 你 + [具体动作/位置]" 结构——极度关注"你"的具体细节而非"我"的情感

## 三、作曲层 — 像陈奕迅一样谱曲

### 3.1 旋律设计规则

**规则 1：旋律服从语气，不服人和弦套路**
- 先念歌词，录下来 → 标记自然音高起伏 → 以此为基础设计旋律线
- 说话的停顿 = 休止符；说话的上扬 = 音阶上行；说话的叹气 = 音阶下行
- 禁止"因为和弦走到了这里所以旋律应该这样走"——和弦为旋律服务，不是反过来

**规则 2：大量使用半音阶和离调**
- 半音阶（小二度进行）是尘式旋律的秘密武器
- 用法：情感犹豫/纠结/不能一步到位时 → 不要直接跳过去，走半音爬过去
- 参考：《不如不见》前奏的半音下行；《富士山下》主歌的离调处理
- 禁止：全程自然音阶的"顺滑"旋律——太顺滑 = 不够纠结 = 不像陈奕迅

**规则 3：真假声剧烈切换 = 情感失控点**
- 副歌安排一次从低音（真声）→ 突然翻高八度（假声/头声）的跳跃
- 这个跳跃放在歌词最"失控"的那个词上
- 参考：《反高潮》的真假声切换；整套"U87"时期作品的极端音域跨越
- 禁止：全程在一个声区平稳进行——没有失控点就没有尘味

**规则 4：句尾下坠 = 叹气感 = 无力感**
- 关键句（尤其是副歌最后一句）的最后一个字做下行
- 不是"解决"到主音，而是"掉下去"到不确定的音
- 参考：《明年今日》"气"字往下掉；《好久不见》每句末尾都在下坠

### 3.2 节奏设计规则（尘式自由节奏）

**核心：打破规整，追求"不完美"的呼吸感**

**主歌节奏：**
- 大量使用**附点、三连音、五连音**——让每个字的时值都不一样
- 参考：《好久不见》"我~~来到~你的城市"——没有一个字的时值相同
- 规则：把歌词当台词念一遍，念的情绪起伏 = 节奏的松紧

**副歌节奏：**
- 突然的**切分重音**——把重拍放在"的、了、吗"等虚词上
- 故意破坏听众的节奏预期
- 高潮处可以出现**抢拍**（比预期早半拍进）或**拖拍**（故意晚半拍唱）

**句尾处理：**
- 每句结束后伴奏空一拍甚至一整小节——留出"沉默的空间"
- 沉默 = 戏剧感。不是所有的空都要填满
- 参考：《好久不见》《不如不见》中大量留白

**节奏禁忌：**
- 禁止全曲节奏规整（那叫儿歌，不叫尘式）
- 禁止节拍器式的完美对齐——允许细微的拖拍和抢拍
- 副歌禁止照搬主歌节奏型——必须出现节奏突变

### 3.3 和弦走向

**基础理念：复杂却"好听"的不协和**

```
调性偏好：降E大调 / C小调（偏好带暗色的调性）

核心走向 1（暗流型）：
I - V7 - vi - iii7 - IV - I/ii - ii7 - V7
加入大量七和弦制造"美好的苦涩"
代表：《富士山下》《不如不见》

核心走向 2（爆发型）：
i - VI - III - VII（小调框架）
突然转向同主音大调 I - IV - V
然后再回到小调
代表：《浮夸》桥段、《反高潮》

关键色彩技���：
- 七和弦（maj7/min7/dom7）：基础款，几乎每小节都有
- 九和弦（add9/9th）：在情感"悬停"处使用
- 增和弦（aug）：在"扭曲感""失控感"处使用
- 同主音大小调切换：希望→绝望的瞬间切换（C→Cm 或 Cm→C）
- 副属和弦：在桥段离调时密集使用，制造"走入迷宫"的感觉
```

**和弦使用铁律：**
- "得体的美"用大七（maj7），"不堪的美"用小七（min7）
- 歌词写到转折时，和弦必须同步变化——不允许歌词走向暗面但和弦还停在亮面

### 3.4 编曲与声部设计

- **钢琴主导**（慢歌）：单音或简约柱式和弦，让人声的每个细节都被听到
- **弦乐铺底**（叙事高潮）：不是从头到尾，而是只在关键段落进入——"该来的时候来"
- **突然安静**：Bridge 段经常只剩钢琴 + 人声，甚至只剩人声 + 呼吸
- **荒诞段落**：允许加入反常元素——歌剧吟唱、一段念白、突然的静默
- **人声处理**：不追求"录音室完美"，保留换气声、轻微的颤抖、甚至走音的边缘感

## 四、创作模板

### 模板 A：尘式叙事慢歌（怀念/释怀类）

```
【第一步：找到一个"不起眼"的日常切入点】
不要写"我想你"，写：
- "地铁站的重逢 + 假装没看到"
- "整理旧物找到没还你的半本漫画"
- "路过那家倒闭的唱片店"
→ 越具体、越琐碎、越"不值得写进歌里"的场景越好

【第二步：主歌用超长句叙事（A1-A2）】
A1：当下的琐碎动作 + 环境细节
    "那个 + [日常场景] + [细节A] + [细节B] + [仿佛什么都没变]"
A2：由场景触发的回忆碎片（不要解释，只呈现）
    "想起 + [具体事件A] / [物件B] / [一句话C]"
→ 每句20-30字，用逗号连成连珠炮，中间不换气

【第三步：预副歌用一个"反转发问"过渡】
"[看似平静的陈述] + [但/竟] + [一个无法回答的问题]"
例如："咖啡还是热的，但我忽然忘了你的口味——是从什么时候开始的？"

【第四步：副歌用"糖衣刀片"金句】
"[美好的画面/回忆] + [转折] + [戳破它的真相]"
→ 副歌第一句必须让听众点头
→ 第二句必须让听众沉默2秒
→ 最高音/最失控的假声放在"真相"那个词上
→ 最后一句的最后一个字下行叹气

【第五步：桥段允许"失控"】
- 可以变得荒诞（突然加快/突然安静/加入念白）
- 可以自嘲（"我知道这样很可笑，但……"）
- 必须在末尾突然收住，回归安静

【第六步：和弦编排】
降E大调 → 大量七和弦 + 半音离调
关键转折处：同主音大小调切换
桥段：副属和弦连接链
结尾：回到 I，但不解决到舒适——挂一个 sus4 不解决

【第七步：节奏编排】
主歌：附点 + 三连音，模拟说话的自然错落
预副歌：逐渐规整，制造"要发生什么了"的预感
副歌：切分重音 + 拖拍 + 关键句末尾留出整小节空白
桥段：节奏可"溃散"，然后再聚拢
```

### 模板 B：尘式爆发曲（浮夸/反高潮型）

```
【第一步：选一个"不正常"的主题】
- 想红的卑微（《浮夸》）
- 不想被期待的窒息感
- 对"正能量"的反胃
→ 主题必须带点"不太正确"的锋利

【第二步：主歌用"微信语音"质感】
超长句 + 口语化到极致 + 允许不雅字眼
内容：描述一个让人尴尬/卑微/不舒服的处境

【第三步：预副歌逐渐加压】
旋律线爬升 + 节奏加密 + 和弦复杂度增加
→ 像一个气球慢慢被吹胀

【第四步：副歌——破音点 + 情感爆炸】
- 在副歌第一句设置"破音点"
- 旋律跳到真假声切换的临界区
- 歌词必须在这个位置喊出"最不想被人知道的真实想法"
- 和弦同步切换到大小调对立

【第五步：结尾不要完美收束】
- 突然结束（不解决到主和弦）
- 或 fade out 中还在喃喃自语
→ 完美收束 = 完美的假象，要打破它
```

## 五、创作禁区

**以下行为会让作品失去陈奕迅风格的核心竞争力：**

1. **禁止：把歌词写得太"美"**
   - 四字成语、华丽辞藻、古风句式——这是禁区中的禁区
   - 替代方案：保留一句"糙"的作为锚点——"那个字我忘了怎么念"

2. **禁止：情感直说**
   - "我好难过""我真的很爱你""你为什么要离开我"——通通禁止
   - 替代方案：写一个动作、一个场景、一个细节，让听的人自己得出结论

3. **禁止：旋律全程顺滑**
   - 自然音阶从头走到尾 = 这不是陈奕迅
   - 必须在关键情感位置加入半音、离调、或大跳

4. **禁止：节奏四平八稳**
   - 如果每个字都在拍子上、每条旋律都工整对齐 → 重写节奏
   - 检验标准：跟着打拍子，如果全程不打磕绊，说明节奏太规整

5. **禁止：真声唱完全曲**
   - 必须有真假声转换的"失控点"——哪怕只有一处
   - 没有真假声切换 = 没有脆弱感 = 没有尘味

6. **禁止：完美的收尾**
   - 不要"终于释怀了""一切都好了"的结局
   - 尘式的结尾是"算了""就这样吧""我也不知道"

7. **禁止：在桥段保持克制**
   - 桥段是唯一可以"过火"的地方——要疯就疯到底
   - 如果桥段和主歌一样克制，等于浪费了陈奕迅最重要的戏剧空间

8. **禁止：用普通话思维写粤语词（或反之）**
   - 国语尘式和粤语尘式的密押逻辑不同
   - 粤语：九声六调自然押韵，密到喘不过气
   - 国语：保留"连珠炮"节奏但放宽韵脚密度

## 六、对比示例

### 示例 1：写怀念旧情人（错误 vs 正确）

❌ **错误写法（唯美煽情型）：**
```
夜深人静的时候我又想起了你
想起我们曾经的海誓山盟
那些美好的回忆像潮水般涌来
如果时光可以倒流该多好
```
> 问题：全是抽象情感词、空无一物、旋律想象一下就是千篇一律

✅ **尘式写法：**
```
[主歌·超长句]
那个 你上次说借我的那本漫画 我其实没看
放在枕头底下压皱了 封面那道折痕 有点像你皱眉的样子
说起来 咖啡的口味也是跟你学的 加两份糖
现在一个人喝 总觉得太甜 但又改不掉

[预副歌]
原来习惯是长在身体里的 烂也烂不掉

[副歌]
我多想和你见一面 只是见一面
看看你是不是也学会 把糖减到一份
然后假装不认识 走过那条街
像两个陌生的老朋友
```
> 改进：漫画/咖啡/糖/折痕——四个极其具体的日常物件；"陌生的老朋友"是典型的尘式悖论金句；整段没有任何"想念""痛苦""爱"的字眼，但每一句都在说想念

### 示例 2：写自卑/被忽视（错误 vs 正确）

❌ **错误写法（自怜型）：**
```
我总是那个被忽略的人
站在聚光灯照不到的角落
没有人知道我也在努力发光
我也渴望被看见
```
> 问题：自怨自艾、没有画面、像日记本里的抱怨

✅ **尘式写法（浮夸精神）：**
```
[主歌]
那年年会 全场都上台了 只剩我坐在角落
领导说小陈你也来啊 我说我去洗手间
在隔间里刷了十分钟朋友圈 把所有人的合影都点了个赞
然后冲水 假装我刚回来

[预副歌]
我开始想 如果我在台上会不会不一样

[副歌·歇斯底里]
你看不见我 不是因为灯光太暗
是我从来就没亮过
我连影子都是别人的影子里 借来的
那就浮夸吧 反正也没人在乎真假
```
> 改进："刷朋友圈点赞""冲水假装刚回来"——极其真实但从来没被写进歌里的细节；副歌"我连影子都是借来的"是典型的"自嘲放大"公式；结尾不说"我要努力发光"，而是"那就浮夸吧"——认了，但认得不甘心

### 示例 3：写时光流逝（错误 vs 正确）

❌ **错误写法（鸡汤型）：**
```
时光匆匆流过 带走了青春
但我依然感谢每一天
珍惜当下 活在当下
未来会更美好
```
> 问题：全部是空洞的正能量短语，毫无信息量

✅ **尘式写法：**
```
[主歌]
上次路过中学门口 保安换了人
以前那个光头大叔 总在校门口抽烟
他的烟灰落到我试卷上 烫了一个洞
那次的物理考了58分 他帮我改成了85

[预副歌]
不知道他是不是退休了 还是换了学校
或者 只是没人记得

[副歌]
多遥远的光头保安 一个下午就走了
当时以为只是放暑假 他说"下学期再见"
后来才知道 有些人说再见 是因为他也不知道
那是最后一面
```
> 改进：不写"时光流逝"，写一个活生生的光头保安；不感慨"时间好快"，写具体的"烟灰烫试卷""58分改85分"；副歌用"以为只是放暑假"这种人人都有过的经验来引爆共鸣——不是"时间好快"的共鸣，是"原来那已经是最后一面了"的后知后觉

## 八、局限性 + 自检清单

### 8.1 局限性声明

1. **作词来源限制**：陈奕迅的歌词主要为林夕、黄伟文等填词人所作，本指南分析的是"陈奕迅歌曲的歌词风格"而非他本人的作词习惯。但作为演唱者，他的选词品味和演绎偏向构成了风格判断的基础。
2. **粤语/国语双轨**：粤语歌和国语歌在语言的密押密度、节奏感上有本质差异。本指南偏重粤语底层逻辑（因其更能代表尘式巅峰），国语应用时需适度放宽韵脚密度。
3. **曲风跨度极大**：从芭乐、爵士、摇滚到电子均涉猎。本指南偏重其最具辨识度的"叙事慢歌 + 戏剧爆发"风格。
4. **样本时代**：主要基于英皇/新艺宝时期作品（2000-2015），后期作品风格更趋成熟内敛。

### 8.2 创作前自检清单

在用这个 skill 创作前，先确认：

- [ ] 我找的切入点是否足够"俗"/"琐碎"？（太宏大 = 不合格）
- [ ] 主歌是否有至少一个"微信语音质感"的超长句？
- [ ] 歌词中是否有至少一句"不太好看但很真实"的表述？
- [ ] 副歌是否完成了先给糖再给刀的结构？
- [ ] 副歌金句是否能让人听完沉默2秒？
- [ ] 旋律是否有半音/离调在关键情感词上？
- [ ] 是否安排了至少一处真假声切换的"失控点"？
- [ ] 是否有至少一处节奏"不规整"（附点/三连音/拖拍/抢拍）？
- [ ] 句尾是否留了足够的空白/沉默空间？
- [ ] 和弦是否有七和弦/九和弦/增和弦的"苦涩感"？
- [ ] 桥段是否做了"失控/荒诞/自嘲"处理？
- [ ] 结尾是否避免了"完美的圆满收束"？
- [ ] 歌词中有没有"永远/一辈子/正能量/一定要幸福"等禁词？
- [ ] 有没有"我好想你/我好难过"之类的直白情感句？有 → 删掉，用动作替代
